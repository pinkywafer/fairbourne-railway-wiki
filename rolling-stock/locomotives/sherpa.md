# Sherpa

**Type:** Steam (Replica Darjeeling B Class)  
**Builder:** Milner Engineering (1978)  
**Gauge:** 12¼ in  
**Status:** In service

Sherpa is a half-size replica of the Darjeeling Himalayan Railway Class B locomotives. Built for the Réseau Guerlédan line in France, later moved to Fairbourne.
