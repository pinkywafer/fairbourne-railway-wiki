# Contributing to Fairbourne Railway Wiki

Contributions are welcome!

## How to contribute
1. Fork this repository.
2. Create a new branch for your edits.
3. Make your changes (fix errors, add photos, update data).
4. Submit a pull request for review.

## Guidelines
- Use Markdown format for all pages.
- Keep filenames lowercase with hyphens.
- Include references or image credits if adding new media.
